# generator-apigee-proxy

This is a code generator for Apigee API proxies built using [Yeoman](https://yeoman.io) and [apigeetool](https://github.com/apigee/apigeetool-node).

## How to use it

    yo apigee-proxy <proxy_name> <base_path> <apigee_username> <apigee_password> <organization> <environment>
